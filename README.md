ShowCase Animation

Интерактивный слайдер для портфолио с эффектом жидкого стекла и плавными переходами.
📋 Описание

Этот проект представляет собой современный веб-слайдер, ориентированный на визуальную подачу контента. Основной акцент сделан на плавности анимаций и использовании современных CSS-эффектов, таких как Glassmorphism.
🛠 Технологии

    HTML5 / CSS3 — верстка и стилизация (Liquid Glass effect).

    JavaScript (ES6+) — логика переключения и управления данными.

    GSAP (GreenSock Animation Platform) — ядро для высокопроизводительных анимаций.

    Google Fonts — типографика (Inter, Montserrat).

✨ Особенности

    Интерактивность: Управление осуществляется с помощью колесика мыши (скролла).

    Адаптивность: Слайдер корректно отображается как на мониторах, так и на мобильных устройствах.

    Динамический текст: У каждой локации есть свое уникальное описание, которое меняется вместе с фоном.

    Оптимизация: Плавная работа даже при большом количестве графических элементов.
