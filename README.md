# ShowcaseAnimation 🚀

Интерактивный слайдер премиум-класса с эффектом жидкого стекла и плавной анимацией.

### Технологии:
* **GSAP** — для сложных переходов и таймингов.
* **CSS Glassmorphism** — эффект матового и жидкого стекла.
* **JavaScript (Custom Scroll API)** — управление контентом через колесо мышки.

### Как запустить:
Просто откройте `index.html` в любом современном браузере.
